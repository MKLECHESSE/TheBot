TROUBLESHOOTING QUICK FIXES
────────────────────────────────────────────────────────────────────────────────

"ModuleNotFoundError: No module named X"
  → pip install -r requirements.txt

"Symbol not found on MT5"
  → Check Market Watch (Ctrl+M in MT5)
  → Use symbol_aliases in config.yaml

"Order rejected: insufficient balance"
  → Increase demo account balance
  → Reduce risk_percentage in config

"Telegram/Email not sending"
  → Verify .env file has credentials
  → For Gmail: use App Password, not account password

"No notifications received"
  → Check .env has correct values
  → Verify Telegram bot token and chat ID
  → Test desktop notification manually

"Bot crashed or won't start"
  → Check Python venv: .venv\Scripts\python.exe --version
  → Check MT5: python -c "import MetaTrader5; print(MetaTrader5.initialize())"
  → View full error: python TheBot.py 2>&1
